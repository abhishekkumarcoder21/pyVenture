# PyVenture

PyVenture ek Python-based adventure game project hai. Isme aapko game logic, sprites, UI, aur settings ke alag-alag modules milenge. Yeh project beginners ke liye Python programming aur game development seekhne ka ek accha mauka hai.

## Features

- Modular code structure (game logic, sprites, UI, settings)
- Easy to extend and customize
- Simple UI for interaction

## Requirements

Sabse pehle, dependencies install karen:

```
pip install -r requirements.txt
```

## Run kaise karein

Game chalane ke liye yeh command use karein:

```
python main.py
```

## File Structure

- `main.py` : Game ka entry point
- `game.py` : Main game logic
- `sprites.py` : Game sprites aur characters
- `ui.py` : User interface
- `settings.py` : Game settings
- `requirements.txt` : Required Python packages
